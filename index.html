ngGallery

AngularJS Gallery Slideshow

Example - Default

Default option is displayed with 3 thumbnails in popup

<ng-gallery images="ctrl.images">
</ng-gallery>

Example - n thumbnails

Example with 11 thumbnails

<ng-gallery images="ctrl.images" thumbs-num="11">
</ng-gallery>

Install

$ bower install ngGallery

Configuration

Include CSS files

<link rel="stylesheet" type="text/css" href="bower_components/src/css/ngGallery.css">
<link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/font-awesome/4.3.0/css/font-awesome.min.css">

Include JavaScript files

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/angularjs/1.4.0/angular.min.js"></script>
<script type="text/javascript" src="bower_components/src/js/ngGallery.js"></script>

Include module in your app

var myApp = angular.module('myApp', ['jkuri.gallery']);

Initialize images, of course you can also do that via $http (Ajax) 

myApp.controller('Ctrl', [function() {
    var self = this;
 
    self.images = [
        {thumb: 'images/thumbs/1.jpg', img: 'images/1.jpg', description: 'Image 1'},
        {thumb: 'images/thumbs/2.jpg', img: 'images/2.jpg', description: 'Image 2'},
        {thumb: 'images/thumbs/3.jpg', img: 'images/3.jpg', description: 'Image 3'},
        {thumb: 'images/thumbs/3.jpg', img: 'images/4.jpg', description: 'Image 4'}
    ];
]);
